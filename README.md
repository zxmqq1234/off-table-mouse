# 🖱️ 桌外鼠标 — 手机秒变电脑无线触控板

**你的手机，就是最好的电脑遥控器。**

没有蓝牙？没有驱动？没有额外硬件？  
只要扫个码，手机立刻变成一块**全功能无线触控板**——鼠标、键盘、手势、快捷键，全在指尖。

---

## 🎬 场景故事

### 场景一：沙发上的「第二屏」

> 你靠在沙发上，电脑连着投影仪放电影。  
> 想暂停？想调音量？想切下一集？  
> 没必要起身摸鼠标——掏出手机，打开浏览器，扫一扫屏幕上的二维码。  
> 手指一滑，电影暂停；双指一拨，音量调节；三指上滑，多任务切换。  
> 全程不离开沙发，全程不打断沉浸在沙发里的姿势。

**桌外鼠标 = 你的无线遥控器。**

### 场景二：会议室的「救场神器」

> 演示到一半，PPT 遥控器没电了。  
> 全场十几号人盯着你。  
> 你从兜里掏出手机，扫码，翻页——行云流水。  
> 激光笔？不需要。回来翻页？双指向左滑。白板批注？长按拖拽。  
> 台下的 IT 总监看呆了。

**桌外鼠标 = 你的应急演示遥控器。**

### 场景三：躺在床上追剧

> 冬天的被窝，笔记本放在床头柜。  
> 你的手不想伸出被窝，更不想去够鼠标。  
> 手机就在枕头边。扫码，连接，然后你只需要在被窝里轻轻滑动——  
> 音量、播放、下一集、全屏、退出。  
> 整个电脑被你的一只手「隔空」掌控。  
> 这才是冬天该有的体验。

**桌外鼠标 = 被窝里的智能遥控器。**

### 场景四：多屏工作台的「效率外挂」

> 你面前两块大屏，鼠标来回跑。  
> 但你发现——鼠标要精准定位很慢，切窗口要点半天。  
> 掏出手机：三指左右滑 = 快速切换窗口；三指上滑 = 任务视图；  
> 三指下滑 = 显示桌面。触控区边缘移过去，指针自动滑到另一块屏。  
> 你的手指变成了鼠标的「加速器」——只在需要精确点击时才用手指，其余全交给手势。

**桌外鼠标 = 你的效率加速器。**

## 🚀 快速开始

### 1. 安装（Windows）

```bash
# 克隆项目
git clone https://gitee.com/ye_zhongji/off-table-mouse.git
cd off-table-mouse

# 安装依赖
npm install
```

### 2. 启动服务

```bash
npm run dev
```

电脑端会启动一个本地 HTTP 服务，并在屏幕右上角显示一个 —— **二维码**。

### 3. 手机扫码连接

用手机浏览器（推荐 Chrome / Edge / Safari）扫描二维码。

> ⚡ 同一局域网下，扫完即连，无需任何配置。

### 4. 开始操控

| 操作 | 效果 |
|------|------|
| 单指滑动 | 鼠标移动 |
| 单指点击 | 鼠标左键单击 |
| 单指双击 | 鼠标左键双击 |
| 单指长按 + 滑动 | 拖拽 |
| 双指点击 | 鼠标右键 |
| 双指左右滑 | 前进 / 返回 |
| 三指上滑 | 多任务视图（Win+Tab） |
| 三指下滑 | 显示桌面（Win+D） |
| 三指左右滑 | 切换窗口（Alt+Tab） |
| 三指点击 | 鼠标中键 |

## 🛠️ 技术栈

| 层 | 技术 |
|----|------|
| 桌面端框架 | Electron 33 |
| 前端框架 | Vue 3 + Vite 5 |
| 实时通信 | WebSocket（ws 库） |
| 真实输入模拟 | nut.js（非模拟，直接操作系统级鼠标/键盘） |
| 多屏感知 | Electron Screen API（每个显示器独立坐标与 DPI） |
| 动效反馈 | 透明 Overlay 窗口 + CSS 动画（涟漪、气泡、滑入） |
